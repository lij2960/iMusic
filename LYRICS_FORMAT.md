# iMusic 歌词文件格式说明

## 支持的歌词格式

### LRC 格式 (.lrc)
```
[00:12.50]第一行歌词
[00:17.20]第二行歌词
[00:21.10]第三行歌词
[01:06.00]副歌部分
```

### 纯文本格式 (.txt)
```
第一行歌词
第二行歌词
第三行歌词
副歌部分
```

## 歌词文件放置位置

应用会按以下顺序查找歌词文件：

1. 与音乐文件同名同目录：`/path/to/song.mp3` -> `/path/to/song.lrc`
2. 同目录下的 lyrics 文件夹：`/path/to/lyrics/song.lrc`
3. 支持 .lrc 和 .txt 两种扩展名

## 使用方法

1. 将歌词文件放在音乐文件同目录下
2. 确保歌词文件名与音乐文件名相同（扩展名除外）
3. 在播放界面点击歌词按钮查看歌词
4. 点击歌词行可以跳转到对应时间点（仅LRC格式）

## 示例

音乐文件：`/storage/emulated/0/Music/我的歌曲.mp3`
歌词文件：`/storage/emulated/0/Music/我的歌曲.lrc`

歌词内容：
```
[00:00.00]我的歌曲
[00:05.00]演唱者：某某
[00:10.00]
[00:15.50]这是第一句歌词
[00:20.30]这是第二句歌词
[00:25.10]这是第三句歌词
```
