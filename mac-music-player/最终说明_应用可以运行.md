## 为什么会觉得"运行不起来"？

可能的原因：

### 1. 窗口不可见
- 窗口可能在其他桌面空间
- 窗口可能最小化
- 窗口位置可能在屏幕外

### 2. 使用open命令后立即退出
这是正常的！`open`命令会：
1. 启动应用
2. 立即返回终端
3. 应用在后台运行

### 3. 没有明显的视觉反馈
PyQt5应用可能不会立即显示在Dock中。

## 正确的使用方法

### 方法1：使用启动脚本（推荐）
```bash
cd mac-music-player
./启动应用.sh
```

这会：
- 检查应用是否已在运行
- 启动应用
- 显示进程信息
- 提供使用提示

### 方法2：从终端运行
```bash
cd mac-music-player
./iMusic.app/Contents/MacOS/iMusic
```

优点：
- 可以看到所有输出
- 方便调试
- 窗口会显示

### 方法3：使用open命令
```bash
open mac-music-player/iMusic.app
```

这是标准的macOS启动方式。

### 方法4：双击运行
在Finder中双击 `iMusic.app`。
