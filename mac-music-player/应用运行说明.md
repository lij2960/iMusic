## 可能的问题

### 1. 窗口没有显示
如果应用运行但看不到窗口，可能是：
- 窗口在其他桌面空间
- 窗口最小化到Dock
- 窗口位置在屏幕外

**解决方法**：
```bash
# 方法1：使用open命令
open iMusic.app

# 方法2：从终端运行
./iMusic.app/Contents/MacOS/iMusic

# 方法3：检查所有窗口
# 使用Mission Control (F3) 查看所有窗口
```

### 2. 应用立即退出
如果使用`open`命令后应用立即退出，可能是：
- 没有选择音乐文件夹
- 数据库初始化失败
- 权限问题

**解决方法**：
```bash
# 从终端运行查看错误
cd mac-music-player
./iMusic.app/Contents/MacOS/iMusic
```

### 3. 首次运行权限
macOS可能阻止未签名的应用。

**解决方法**：
1. 右键点击 `iMusic.app`
2. 选择"打开"
3. 在弹出对话框中点击"打开"

## 推荐运行方式

### 方式1：从终端运行（推荐用于测试）
```bash
cd mac-music-player
./test_run.sh
```

这会：
- 启动应用
- 显示进程信息
- 保存日志到 `app_run.log`

### 方式2：直接运行
```bash
cd mac-music-player
./iMusic.app/Contents/MacOS/iMusic
```

这会在终端显示所有输出，方便调试。

### 方式3：使用open命令
```bash
open mac-music-player/iMusic.app
```

这是正常的macOS应用启动方式。

### 方式4：双击运行
直接在Finder中双击 `iMusic.app`。

## 常见问题

### Q: 应用运行但看不到窗口
A: 尝试以下方法：
1. 按 Command+Tab 切换到iMusic
2. 按 F3 查看所有窗口
3. 检查Dock中是否有iMusic图标
4. 从终端运行查看输出

### Q: 应用立即退出
A: 从终端运行查看错误信息：
```bash
./iMusic.app/Contents/MacOS/iMusic
```

### Q: 提示"无法打开"
A: 右键点击 > 打开，或者在系统偏好设置中允许。

### Q: 没有音乐显示
A: 确保：
1. 已选择包含音乐文件的文件夹
2. 音乐文件格式支持（MP3, M4A, FLAC等）
3. 文件有读取权限
