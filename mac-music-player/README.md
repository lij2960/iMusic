## 功能特性

### 1. 音乐管理
- ✅ 指定目录导入本地音乐文件
- ✅ 支持多种音频格式（MP3, WAV, FLAC, AAC, OGG, M4A, WMA）
- ✅ 自动提取音乐元数据（标题、艺术家、专辑、时长）
- ✅ 自动加载专辑封面

### 2. 排序功能
- ✅ 默认按文件创建日期排序（最新优先）
- ✅ 支持8种排序方式：
  - 按日期（升序/降序）
  - 按标题（A-Z/Z-A）
  - 按艺术家（A-Z/Z-A）
  - 按时长（短/长）

### 3. 播放模式
- ✅ 列表循环播放（默认）
- ✅ 单曲循环
- ✅ 随机播放

### 4. 歌词功能
- ✅ 自动检测同目录下的歌词文件（.lrc, .txt）
- ✅ 实时同步显示歌词
- ✅ 支持点击歌词跳转到对应时间点
- ✅ 提供完整歌词查看对话框
- ✅ 当前播放行高亮显示

### 5. 播放状态缓存
- ✅ 记住上次播放的歌曲
- ✅ 保存播放位置
- ✅ 保存播放模式和排序设置
- ✅ 应用重启后自动恢复播放状态

### 6. 音质优化
- ✅ 使用VLC引擎进行高质量音频播放
- ✅ 支持多种音频格式和编解码器
- ✅ 专业级音频处理
- ✅ 立体声输出

## 安装说明

### 1. 环境要求
- macOS 10.14 或更高版本（已测试macOS 12.7.6 ✅）
- Python 3.8 或更高版本
- **VLC媒体播放器** （必需）
- pip 包管理器

### 2. 安装VLC

本程序使用VLC作为音频播放引擎，需要先安装VLC：

**方法1: 使用Homebrew（推荐）**
```bash
brew install --cask vlc
```

**方法2: 手动下载**
访问 [VLC官网](https://www.videolan.org/vlc/) 下载并安装

### 3. 安装Python依赖

```bash
# 进入项目目录
cd mac-music-player

# 运行安装脚本（会自动检查VLC并安装依赖）
./setup.sh
```

或者手动安装：

```bash
# 创建虚拟环境（推荐）
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 4. 运行程序

```bash
# 使用运行脚本
./run.sh
```

或者手动运行：

```bash
# 确保在虚拟环境中
source venv/bin/activate
python main.py
```

## 使用指南

### 1. 添加音乐
1. 点击"添加音乐文件夹"按钮
2. 选择包含音乐文件的文件夹
3. 程序会自动扫描并导入所有支持的音频文件

### 2. 播放音乐
- 双击歌曲列表中的歌曲开始播放
- 使用播放控制按钮：播放/暂停、上一首、下一首
- 拖动进度条调整播放位置
- 调整音量滑块控制音量

### 3. 排序和播放模式
- 使用顶部的"排序"下拉菜单选择排序方式
- 使用"播放模式"下拉菜单切换播放模式

### 4. 歌词显示
- 将歌词文件（.lrc或.txt格式）放在音乐文件同一目录下
- 歌词文件名需与音乐文件名相同（例如：song.mp3 对应 song.lrc）
- 播放时会自动加载并同步显示歌词
- 点击歌词行可跳转到对应时间点
- 点击"完整歌词"按钮查看完整歌词内容

### 5. 歌词格式

#### LRC格式（推荐）
```
[00:12.50]第一行歌词
[00:17.20]第二行歌词
[00:21.10]第三行歌词
```

#### 纯文本格式
```
第一行歌词
第二行歌词
第三行歌词
```

## 快捷键

- **空格键** - 播放/暂停（计划中）
- **→** - 下一首（计划中）
- **←** - 上一首（计划中）
- **↑** - 增加音量（计划中）
- **↓** - 减少音量（计划中）

## 注意事项

1. **音频格式支持**
   - 确保音频文件格式受支持
   - MP3格式兼容性最好
   - FLAC格式提供无损音质

2. **歌词文件**
   - 歌词文件必须与音乐文件在同一目录
   - 文件名必须完全匹配（不含扩展名）
   - 推荐使用LRC格式以获得时间同步

3. **性能优化**
   - 首次扫描大量音乐文件可能需要一些时间
   - 数据库会缓存音乐信息，后续启动更快
   - 建议定期清理不存在的音乐文件

4. **数据存储**
   - 数据库文件保存在程序运行目录
   - 文件名：`music_player.db`
   - 可以手动删除以重置所有数据

## 故障排除

### 问题：安装时提示找不到VLC
- 确保已安装VLC媒体播放器
- 使用 `brew install --cask vlc` 安装
- 或从 https://www.videolan.org/vlc/ 下载安装

### 问题：无法播放音乐
- 检查VLC是否正确安装
- 检查音频文件是否损坏
- 确认文件格式是否支持
- 尝试重新导入音乐文件

### 问题：歌词不显示
- 确认歌词文件存在
- 检查歌词文件名是否与音乐文件匹配
- 验证歌词文件格式是否正确

### 问题：程序无法启动
- 确认Python版本 >= 3.8
- 检查所有依赖是否正确安装
- 尝试重新安装依赖包
