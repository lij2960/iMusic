# iMusic 完整打包流程

## 🎯 目标

从源代码到可分发的 DMG 安装包

## 📋 完整流程

### 第一步：准备环境

```bash
cd mac-music-player

# 创建 Python 3.12 环境
bash setup_python312.sh
```

**输出**：
- Python 3.12.8 环境
- 虚拟环境 `venv312/`
- 所有依赖已安装

### 第二步：打包应用

```bash
# 激活环境（如果需要）
source venv312/bin/activate

# 打包应用
bash create_app_py312.sh
```

**输出**：
- `iMusic_py312.app` (180MB)
- 包含所有依赖
- 可以独立运行

**耗时**：约 2-3 分钟

### 第三步：测试应用

```bash
# 运行测试脚本
bash test_final.sh
```

**测试内容**：
- ✓ 应用启动
- ✓ 窗口显示
- ✓ 扫描音乐
- ✓ 播放音乐
- ✓ 搜索歌词

### 第四步：创建 DMG

```bash
# 创建 DMG 安装包
bash create_dmg_py312.sh
```

**输出**：
- `iMusic-1.0.1-py312.dmg` (152MB)
- 包含安装说明
- 可以直接分发

**耗时**：约 1-2 分钟

### 第五步：验证 DMG

```bash
# 验证 DMG 完整性
hdiutil verify iMusic-1.0.1-py312.dmg

# 测试打开 DMG
open iMusic-1.0.1-py312.dmg
```

## ✅ 完成！

现在你有了：

1. **iMusic_py312.app** - 应用程序
2. **iMusic-1.0.1-py312.dmg** - 安装包

可以分发给用户了！

## 📝 用户安装说明（给用户）

```
iMusic 安装步骤：

1. 下载 iMusic-1.0.1-py312.dmg
2. 双击打开 DMG 文件
3. 将 iMusic.app 拖到 Applications 文件夹
4. 打开 Applications 文件夹
5. 右键点击 iMusic.app，选择"打开"
6. 点击"打开"确认（仅首次需要）
7. 点击"扫描音乐"选择音乐文件夹
8. 开始使用！

系统要求：
- macOS 10.13 或更高版本
- 无需安装 Python 或其他依赖
- 无需联网（除了搜索歌词功能）

注意事项：
- 首次运行必须右键打开
- 之后可以直接双击运行
- 数据保存在 ~/Library/Application Support/iMusic/
```

## ⚠️ 常见问题

### Q: DMG 太大了，能减小吗？
A: 可以，但需要移除一些依赖或使用更激进的压缩。当前大小（152MB）对于包含完整 Python 环境的应用来说是合理的。

### Q: 用户说无法打开应用？
A: 确保用户使用"右键 → 打开"的方式首次运行。

### Q: 如何避免安全警告？
A: 需要 Apple Developer 账号进行代码签名和公证（$99/年）。

### Q: 可以在其他 Mac 上运行吗？
A: 可以！DMG 包含所有依赖，可以在任何 macOS 10.13+ 的 Mac 上运行。
