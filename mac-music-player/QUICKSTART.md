# 快速入门指南

## 前置要求

### 安装VLC媒体播放器

本程序需要VLC作为音频播放引擎，请先安装：

**使用Homebrew（推荐）：**
```bash
brew install --cask vlc
```

**手动下载：**
访问 https://www.videolan.org/vlc/ 下载并安装

## 一键安装和运行

### 1. 安装

在终端中执行：

```bash
cd mac-music-player
./setup.sh
```

这将自动：
- 检查VLC是否安装
- 创建Python虚拟环境
- 安装所有依赖包
- 配置运行环境

### 2. 运行

```bash
./run.sh
```

或者手动运行：

```bash
source venv/bin/activate
python main.py
```

## 首次使用

### 步骤1：添加音乐
1. 点击窗口顶部的"添加音乐文件夹"按钮
2. 选择包含音乐文件的文件夹
3. 等待扫描完成

### 步骤2：播放音乐
1. 在左侧歌曲列表中双击任意歌曲
2. 音乐开始播放，右侧显示播放信息

### 步骤3：添加歌词（可选）
1. 准备歌词文件（.lrc或.txt格式）
2. 将歌词文件放在音乐文件同一目录
3. 确保文件名相同（例如：song.mp3 对应 song.lrc）
4. 重新播放歌曲，歌词会自动显示

## 常用操作

### 播放控制
- **播放/暂停**: 点击"▶ 播放"或"⏸ 暂停"按钮
- **上一首**: 点击"⏮ 上一首"按钮
- **下一首**: 点击"下一首 ⏭"按钮
- **跳转**: 拖动进度条到指定位置
- **音量**: 调整音量滑块

### 排序和播放模式
- **排序**: 使用顶部"排序"下拉菜单
  - 日期添加(最新) - 默认
  - 日期添加(最旧)
  - 标题(A-Z)
  - 标题(Z-A)
  - 艺术家(A-Z)
  - 艺术家(Z-A)
  - 时长(短)
  - 时长(长)

- **播放模式**: 使用"播放模式"下拉菜单
  - 列表循环 - 默认
  - 单曲循环
  - 随机播放

### 歌词功能
- **查看歌词**: 右侧歌词区域实时显示
- **跳转**: 点击歌词行跳转到对应时间点
- **完整歌词**: 点击"📄 完整歌词"按钮查看完整内容

## 歌词文件示例

### LRC格式（推荐）

创建文件 `song.lrc`：

```
[00:00.00]歌曲名 - 艺术家
[00:05.50]第一句歌词
[00:10.20]第二句歌词
[00:15.80]第三句歌词
[00:20.30]第四句歌词
```

### 纯文本格式

创建文件 `song.txt`：

```
第一句歌词
第二句歌词
第三句歌词
第四句歌词
```

## 支持的音频格式

- MP3 (.mp3)
- WAV (.wav)
- FLAC (.flac)
- AAC (.aac)
- OGG (.ogg)
- M4A (.m4a)
- WMA (.wma)

## 常见问题

### Q: 程序无法启动？
A: 
1. 确保已安装VLC媒体播放器
2. 确保已安装Python 3.8+
3. 运行了 `./setup.sh`

### Q: 提示找不到VLC？
A: 
1. 使用 `brew install --cask vlc` 安装VLC
2. 或从 https://www.videolan.org/vlc/ 手动下载安装
3. 安装后重新运行 `./setup.sh`

### Q: 音乐无法播放？
A: 
1. 检查VLC是否正确安装
2. 检查音频文件格式是否支持
3. 尝试使用MP3格式

### Q: 歌词不显示？
A: 确认歌词文件名与音乐文件名完全匹配（不含扩展名）

### Q: 如何重置程序？
A: 删除 `music_player.db` 文件，重新启动程序

## 提示和技巧

1. **批量添加音乐**: 选择包含多个子文件夹的根目录，程序会递归扫描所有音乐文件

2. **专辑封面**: 在音乐文件夹中放置名为 `cover.jpg`、`folder.jpg` 或 `album.jpg` 的图片

3. **播放状态**: 程序会自动记住上次播放的歌曲和位置，下次启动时继续播放

4. **歌词同步**: 使用LRC格式歌词可以获得精确的时间同步效果

5. **音质优化**: 推荐使用FLAC格式获得最佳音质
