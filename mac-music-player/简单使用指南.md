# 简单使用指南

## PyInstaller打包失败？没关系！

PyInstaller在某些环境下可能有兼容性问题。我们提供了更简单可靠的方案。

## 🎯 推荐方案

### 方案1: 桌面快捷方式（最简单）⭐⭐⭐⭐⭐

**一键创建**:
```bash
./create_shortcut.sh
```

**使用**:
- 双击桌面上的"音乐播放器.command"
- 应用在终端窗口中启动
- 关闭终端窗口即可退出

**优点**:
- ✅ 100%可靠
- ✅ 一键创建
- ✅ 双击运行
- ✅ 可以看到日志
- ✅ 无需打包

### 方案2: .app包装器（看起来专业）⭐⭐⭐⭐

**创建**:
```bash
./create_app.sh
```

**使用**:
- 双击"音乐播放器.app"
- 或拖到Applications文件夹
- 像普通macOS应用一样使用

**优点**:
- ✅ 标准.app格式
- ✅ 可以放到Applications
- ✅ 看起来专业
- ✅ 无需打包

**注意**:
- ⚠️ 依赖项目文件夹
- ⚠️ 不能移动项目位置

### 方案3: 直接运行（开发调试）⭐⭐⭐⭐⭐

**运行**:
```bash
./run.sh
```

**优点**:
- ✅ 最简单
- ✅ 最快速
- ✅ 易于调试

## 🚀 快速开始

### 第一次使用

1. **创建桌面快捷方式**
   ```bash
   ./create_shortcut.sh
   ```

2. **双击运行**
   - 在桌面找到"音乐播放器.command"
   - 双击启动

3. **开始使用**
   - 添加音乐文件夹
   - 播放音乐
   - 享受！

### 给朋友使用

如果要分享给朋友：

1. **打包整个项目文件夹**
   ```bash
   cd ..
   zip -r iMusic-mac.zip mac-music-player
   ```

2. **发送给朋友**
   - 发送 `iMusic-mac.zip`
   - 告诉朋友解压后运行 `./create_shortcut.sh`

3. **朋友使用**
   - 解压文件
   - 运行 `./create_shortcut.sh`
   - 双击桌面快捷方式

## ❓ 常见问题

### Q: 为什么PyInstaller打包失败？

A: PyInstaller在Python 3.14和某些macOS版本上有兼容性问题。使用我们提供的简单方案更可靠。

### Q: 桌面快捷方式和.app有什么区别？

A: 
- 桌面快捷方式：在终端窗口中运行，可以看到日志
- .app包装器：像普通应用一样运行，看起来更专业

### Q: 可以移动项目文件夹吗？

A: 
- 桌面快捷方式：移动后需要重新运行 `./create_shortcut.sh`
- .app包装器：移动后需要重新运行 `./create_app.sh`

### Q: 需要安装什么？

A: 
- Python 3.12+
- VLC播放器（`brew install --cask vlc`）
- 项目依赖（已通过 `./setup.sh` 安装）

### Q: 如何卸载？

A: 
- 删除桌面快捷方式
- 删除.app文件
- 删除项目文件夹

## 💡 推荐

**个人使用**: 使用桌面快捷方式（方案1）
**分享给他人**: 使用.app包装器（方案2）
**开发调试**: 直接运行（方案3）
