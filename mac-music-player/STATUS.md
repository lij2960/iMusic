## 🎵 使用流程

### 1. 启动程序
```bash
./run.sh
```

### 2. 添加音乐
- 点击"添加音乐文件夹"按钮
- 选择包含音乐的文件夹
- 等待扫描完成

### 3. 播放音乐
- 双击歌曲列表中的任意歌曲
- 使用播放控制按钮
- 调整音量和进度

### 4. 添加歌词（可选）
- 创建与音乐文件同名的.lrc文件
- 放在同一目录
- 格式: `[00:12.50]歌词内容`

## 🔧 技术细节

### 架构
- **模式**: MVC (Model-View-Controller)
- **语言**: Python 3.8+
- **GUI**: PyQt5 (兼容macOS 12)
- **音频**: VLC引擎
- **数据库**: SQLite

### 支持格式
- **音频**: MP3, FLAC, WAV, AAC, OGG, M4A, WMA
- **歌词**: LRC (带时间戳), TXT (纯文本)
