# iMusic macOS版 - 快速开始

## 运行应用

### 已打包版本（推荐）
如果已经有 `iMusic.app`：
```bash
open iMusic.app
```

### 开发模式
```bash
./run.sh
```

## 打包应用

### 首次打包
1. 确保已安装VLC.app到Applications文件夹
2. 运行打包脚本：
```bash
./create_app.sh
```

### 打包时间
约3-5分钟

### 打包结果
- 应用名称：`iMusic.app`
- 应用大小：约150MB
- 包含：Python运行时 + 所有依赖 + VLC库

## 安装到系统

```bash
cp -R iMusic.app /Applications/
open /Applications/iMusic.app
```

## 创建安装包

```bash
./create_dmg.sh
```

这将创建一个DMG文件，可以分发给其他用户。

## 常见问题

### Q: 应用无法打开
A: 首次运行需要右键点击 > 打开

### Q: 提示"无法验证开发者"
A: 系统偏好设置 > 安全性与隐私 > 允许打开

### Q: 打包失败
A: 确保已安装VLC.app：
```bash
ls /Applications/VLC.app
```

### Q: 需要重新打包
A: 修改代码后运行：
```bash
./create_app.sh
```

## 脚本说明

- `run.sh` - 开发模式运行
- `setup.sh` - 安装依赖
- `create_app.sh` - 打包应用（推荐）
- `create_app_fixed.sh` - 打包应用（备用）
- `create_app_debug.sh` - 打包调试版本
- `create_dmg.sh` - 创建DMG安装包
- `cleanup_old_apps.sh` - 清理旧版本

## 功能特性

✅ 本地音乐播放
✅ 在线歌词搜索
✅ 专辑封面显示
✅ 播放列表管理
✅ 随机播放/循环播放
✅ 音量控制
✅ 进度条拖动

## 技术栈

- Python 3.14
- PyQt5 (GUI)
- VLC (音频播放)
- Mutagen (音频元数据)
- Requests (网络请求)
- SQLite (数据库)

## 系统要求

- macOS 10.13 或更高版本
- VLC.app（打包时需要）
- 约200MB磁盘空间（开发环境）
- 约150MB磁盘空间（打包后应用）
