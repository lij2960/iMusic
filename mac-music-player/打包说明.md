# 打包说明

## 当前状态

打包正在后台进行中，请等待 3-5 分钟。

## 完成后的步骤

### 1. 检查打包是否完成

```bash
cd mac-music-player
ls -lh iMusic_py312.app
```

如果看到 `iMusic_py312.app` 目录，说明打包成功。

### 2. 创建 DMG 安装包

```bash
bash create_dmg_py312.sh
```

### 3. 验证 DMG

```bash
ls -lh iMusic-1.0.1-py312.dmg
```

## 新功能确认

打包后的应用包含以下新功能：

1. ✅ 圆角图标（160x160，应用内和系统图标都是圆角）
2. ✅ 搜索框（在顶部控制栏，播放模式右边）
   - 输入歌曲名或艺术家名实时过滤
   - 不区分大小写
   - 播放列表不受搜索影响

## 测试方法

### 测试搜索功能

1. 打开应用
2. 添加音乐文件夹
3. 在"搜索:"输入框输入关键词
4. 列表实时过滤显示匹配的歌曲

### 测试圆角图标

1. 查看 Dock 中的应用图标
2. 查看 Finder 中的应用图标
3. 应用内播放器页面的专辑封面也是圆角

## 如果打包失败

检查错误信息：

```bash
tail -100 build/iMusic/warn-iMusic.txt
```

重新打包：

```bash
rm -rf build dist iMusic_py312.app *.spec hooks
bash create_app_py312.sh
```
