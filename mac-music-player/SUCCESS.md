# ✅ 安装成功！

恭喜！Python依赖已成功安装。

## 下一步

### 1. 安装VLC（如果还没安装）

VLC是必需的音频播放引擎：

```bash
brew install --cask vlc
```

或访问 https://www.videolan.org/vlc/ 手动下载安装

### 2. 运行程序

```bash
./run.sh
```

或者：

```bash
source venv/bin/activate
python main.py
```

## 首次使用

1. 程序启动后，点击"添加音乐文件夹"
2. 选择包含音乐文件的文件夹
3. 双击歌曲开始播放
4. 享受音乐！🎵

## 功能亮点

- ✅ 支持MP3, FLAC, WAV等多种格式
- ✅ 自动加载歌词（.lrc格式）
- ✅ 实时歌词同步显示
- ✅ 多种排序和播放模式
- ✅ 记住播放状态
