# iMusic 快速使用指南

## 🚀 立即使用

```bash
cd mac-music-player
open iMusic_py312.app
```

就这么简单！应用已经打包好了。

## 📋 功能测试清单

启动应用后，请测试：

1. ✓ 点击"扫描音乐"，选择音乐文件夹
2. ✓ 双击歌曲开始播放
3. ✓ 使用播放控制按钮（播放/暂停/上一首/下一首）
4. ✓ 拖动进度条跳转
5. ✓ 点击"搜索歌词"获取歌词
6. ✓ 查看歌词同步滚动
7. ✓ 切换播放模式（列表循环/单曲循环/随机）
8. ✓ 更改排序方式

## 🔧 重新打包（如需修改代码）

```bash
# 1. 激活 Python 3.12 环境
source venv312/bin/activate

# 2. 重新打包
bash create_app_py312.sh

# 3. 测试
bash test_final.sh
```

## 📦 创建 DMG 安装包

```bash
bash create_dmg.sh
```

这将创建一个可分发的 DMG 文件。

## 🐛 调试

如果遇到问题，从终端运行查看错误信息：

```bash
./iMusic_py312.app/Contents/MacOS/iMusic
```

## 📁 数据位置

- **数据库**: `~/Library/Application Support/iMusic/music_player.db`
- **应用**: `./iMusic_py312.app`

## 💡 提示

- 首次运行需要扫描音乐文件夹
- 歌词文件需要与音乐文件同名（.lrc 或 .txt）
- 支持的音频格式: MP3, WAV, FLAC, AAC, OGG, M4A, WMA
