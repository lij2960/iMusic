## 使用方法

1. 播放一首歌曲
2. 点击"🔍 搜索歌词"按钮
3. 等待搜索完成
4. 如果找到多个版本，选择合适的版本
5. 歌词会自动保存到音乐文件同目录下的.lrc文件
6. 歌词会立即加载并显示

## 与Android版本的一致性

✅ 使用相同的API端点（music.163.com）
✅ 相同的搜索逻辑
✅ 相同的歌词保存位置（音乐文件同目录）
✅ 相同的文件命名规则（歌曲名.lrc）

## 测试步骤

1. 安装依赖（如果还没安装）：
```bash
pip install requests==2.31.0
```

2. 运行程序：
```bash
./run.sh
```

3. 添加音乐文件夹

4. 播放一首没有歌词的歌曲

5. 点击"🔍 搜索歌词"按钮测试在线搜索功能

## 注意事项

- 需要网络连接才能搜索在线歌词
- API可能会有访问限制，如果搜索失败请稍后重试
- 歌词会保存为UTF-8编码的.lrc文件
