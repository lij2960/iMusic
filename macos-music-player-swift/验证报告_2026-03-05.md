## 问题排查

### 如果歌词搜索仍然失败

1. **检查网络连接**
   ```bash
   curl -I https://music-api.heheda.top/
   ```

2. **测试搜索API**
   ```bash
   curl "https://music-api.heheda.top/search?keywords=test&type=1&limit=1"
   ```

3. **查看详细日志**
   - 在Xcode中运行应用
   - 打开Console (Cmd+Shift+Y)
   - 点击"搜索歌词"按钮
   - 查看所有日志输出

4. **检查权限**
   - 确认 `MusicPlayer.entitlements` 中有 `com.apple.security.network.client`
   - 确认应用有网络访问权限

### 如果控制按钮仍然被遮挡

1. **确认代码已更新**
   ```bash
   grep "size: 40" MusicPlayer/Views/PlayerView.swift
   ```

2. **清理并重新编译**
   ```bash
   rm -rf ~/Library/Developer/Xcode/DerivedData/MusicPlayer-*
   xcodebuild -project MusicPlayer.xcodeproj -scheme MusicPlayer -configuration Release clean build
   ```

3. **调整窗口大小**
   - 尝试拖动窗口边缘调整大小
   - 检查不同窗口尺寸下的显示效果
