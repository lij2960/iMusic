# Swift 版 macOS 音乐播放器 - 快速开始指南

## 🚀 5 分钟快速启动

### 步骤 1: 打开项目

```bash
cd macos-music-player-swift
open MusicPlayer.xcodeproj
```

### 步骤 2: 运行项目

在 Xcode 中按 `Cmd+R` 或点击运行按钮 ▶️

### 步骤 3: 使用应用

1. 点击"添加音乐文件夹"
2. 选择包含音乐文件的文件夹
3. 双击歌曲开始播放
4. 享受音乐！🎵

## 📋 系统要求

- macOS 12.0+
- Xcode 15.0+（仅开发需要）

## 🔧 命令行构建（可选）

```bash
./build.sh
```

构建完成后运行：
```bash
open build/Build/Products/Release/MusicPlayer.app
```

## ❓ 常见问题

### Q: Xcode 报错找不到文件？
A: 确保所有文件都在正确的位置，可以尝试 Clean Build Folder (Cmd+Shift+K)

### Q: 无法访问音乐文件夹？
A: 检查系统偏好设置 > 安全性与隐私 > 文件和文件夹，确保应用有访问权限

### Q: 歌曲无法播放？
A: 确保音频文件格式被 AVFoundation 支持（MP3, M4A, FLAC 等）
