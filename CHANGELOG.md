# 更新日志

  - 显示专辑封面
  - 播放/暂停控制

  - 点击跳转到应用
- ✨ 锁屏界面媒体控制
- ✨ 后台播放支持

### 问题修复
- 🐛 修复歌词不跟随播放进度的问题
- 🐛 修复默认封面显示问题
- 🐛 修复播放器封面显示逻辑
- 🐛 修复通知栏在国产手机上的兼容性问题

### 优化改进
- 🔧 优化播放列表管理，支持完整列表加载
- 🔧 优化ExoPlayer配置，支持循环播放
- 🔧 优化通知栏更新机制，实时同步播放状态
- 🔧 移除调试日志，提升运行性能
- 🔧 完善红米、小米等国产手机兼容性

- 使用MediaStyle通知替代PlayerNotificationManager

- 添加前台服务支持

## [1.0.0] - 2024-01-28

### 首次发布
- 🎉 本地音乐播放功能
- 🎉 歌词显示和同步
- 🎉 在线歌词搜索（网易云音乐API）
- 🎉 在线封面搜索
- 🎉 音频均衡器
- 🎉 多种播放模式（顺序/随机/单曲循环）
- 🎉 Material Design 3 界面
- 🎉 自定义进度条
- 🎉 播放状态缓存
- 🎉 搜索功能
- 🎉 放屁网音乐搜索

### 核心功能
- 本地音乐扫描和管理
- 高质量音频播放（ExoPlayer）
- LRC歌词解析和显示
- 专辑封面显示
- 多种排序方式
- 歌曲删除功能

### 技术架构
- MVVM架构
- Jetpack Compose UI
- Hilt依赖注入
- Room数据库
- Retrofit网络请求
- Coroutines异步处理
