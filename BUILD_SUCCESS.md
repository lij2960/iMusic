## 📱 安装和运行

### 方法1：使用Android Studio
1. 在Android Studio中打开项目
2. 连接Android设备或启动模拟器
3. 点击运行按钮 (绿色三角形)

### 方法2：使用命令行
```bash
# 安装到连接的设备
cd /Volumes/Jackey/iMusic
./gradlew installDebug

# 或者手动安装APK
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 🎵 应用功能

### 已实现的核心功能
1. **音乐导入** - 选择目录扫描音乐文件
2. **音乐播放** - 播放/暂停/上一首/下一首
3. **播放模式** - 列表循环/单曲循环/随机播放
4. **搜索功能** - 按歌名、艺术家搜索
5. **排序功能** - 多种排序方式
6. **歌词显示** - 支持LRC格式歌词
7. **状态缓存** - 记住播放状态

### 界面说明
- **音乐库** (底部导航第一个) - 显示所有音乐
- **播放器** (底部导航第二个) - 播放控制界面  
- **导入** (底部导航第三个) - 选择音乐目录

## 🚀 首次使用步骤

1. **授予权限**
   - 首次启动会请求存储权限
   - 点击"授予权限"按钮

2. **导入音乐**
   - 点击底部"导入"标签
   - 选择"全部扫描"或选择特定目录
   - 等待扫描完成

3. **开始播放**
   - 回到"音乐库"标签
   - 点击任意歌曲开始播放
   - 切换到"播放器"标签查看播放界面

## 🎤 歌词功能使用

1. 将歌词文件放在音乐文件同目录下
2. 歌词文件名与音乐文件名相同（扩展名为.lrc或.txt）
3. 播放音乐时点击歌词按钮查看
4. 支持点击歌词行跳转到对应时间点

### 歌词格式示例
```
[00:12.50]第一行歌词
[00:17.20]第二行歌词
[00:21.10]第三行歌词
```

## 🔧 技术特性

- **现代架构**: MVVM + Jetpack Compose + Hilt
- **高质量音频**: ExoPlayer音频引擎
- **本地数据库**: Room数据库存储
- **权限适配**: Android 13+ 媒体权限
- **Material Design 3**: 现代化UI设计

## 📋 注意事项

- 需要Android 7.0 (API 24) 及以上版本
- 首次使用需要授予存储权限
- 歌词文件需要UTF-8编码
- 支持的音频格式：MP3, WAV, FLAC, AAC, OGG, M4A, WMA, OPUS
